# AzsharaSZT
深圳图书馆网络信息空间和创客空间抢位置

![image-20220418235632281](https://cdn.jsdelivr.net/gh/ybm911/blog_picture/img/image-20220418235632281.png)



> 网站预警：
>
> * 密码登陆时用的 get 参数传输明文密码
> * 读取座位数的时候会获取大量用户详细信息，包括密码

开放时间：09:00-21:15（除了周一）

> 每个人的上机次数为4次

创客空间座位数：20

网络信息空间座位数：118

选座逻辑：优先选择创客空间

座位状态：`status` 

* U 使用中
* C 可以使用
* S 无法使用

房间信息：`servaddr`

* `STRead-4FN` 创客空间
* `STRead-4E` 网络信息空间
